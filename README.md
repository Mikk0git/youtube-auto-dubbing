### Simple tool for dubbing youtube videos with AI generatied voice

## Setup

1. Clone repo

```bash
git clone git@github.com:Mikk0git/youtube-auto-dubbing.git
```

2. Download dependencies

```bash
pip install -r requirements.txt
```

3. Run

```python
python main.py
```

- Python
